# NURA — Concept Brand Website

NURA is a modern product landing page concept built to showcase frontend development skills.
The project focuses on strong typography, smooth animations, and a clean editorial layout inspired by modern brand websites.

## Preview

A visually-driven single-page experience exploring the idea of natural fruit candy with playful design and modern interactions.

## Tech Stack

- React
- TypeScript
- Vite
- Framer Motion
- Tailwind CSS
- shadcn/ui

## Features

- Smooth scroll animations
- Editorial typography layout
- Animated product showcase
- Responsive design
- Modern brand-style landing page
